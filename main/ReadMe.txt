Codes complets des différentes parties pour une carte Arduino.

Code initialement prévue pour une seule carte. Il n'a pas été
tester sur le robot.

Se compile avec l'IDE d'Arduino.

Fichiers:
- main.ino : Déclaration des pins et fonction loop et setup.
- fonction.h : Déclaration des méthodes.
- batterie.ino : États de la batterie 
- verrou.ino : verrouillage automatique de la porte.
- balance.ino : Poids des bagages.
- bluetooth.ino : Communication bluetooth avec l'application.
- moteur.ino : gestion des moteurs.
- ultrason.ino : Distance de sécurité avec le fauteuil.
- detection-capteur_infrarouge.ino : buzzer avec la distance de sécurité
